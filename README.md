# Ruinborne

> A roguelike dungeon crawler built in GameMaker Studio using GameMaker Language (GML).

---

## Overview

Ruinborne is an early-stage roguelike dungeon crawler built as a high school programming class project. Players descend through dungeon levels, battling enemies and navigating danger at every turn. The project was a team exploration of game development fundamentals — from game loop architecture and collision systems to enemy AI and room generation — all implemented from scratch in GML.

---

## Tech Stack

| Tool | Purpose |
|------|---------|
| **GameMaker Studio 2** | Game engine and IDE |
| **GameMaker Language (GML)** | Primary scripting language (event-driven, C/JS-style syntax) |

---

## Features (Current Prototype)

- **Player movement & combat** — Responsive input handling with basic melee combat
- **Enemy AI** — Simple state-machine driven enemies (idle, chase, attack)
- **Room-based level design** — Hand-crafted rooms with dungeon traversal logic
- **Health & damage system** — Player and enemy HP tracking with hit feedback
- **Game loop architecture** — Clean separation of game states (menu, play, death, restart)

---

## Project Goals

This project was built to deepen my understanding of:

- **Event-driven programming** in a game engine context
- **State machines** for managing player and enemy behavior
- **Collision detection** and physics handling without a physics engine
- **Rapid iteration** on game feel — tweaking numbers and systems to make gameplay satisfying

---

## Status

🚧 **Early Prototype** — Core systems are functional. Active development ongoing. Not yet in a playable release state.

---

## What I Learned

Working on Ruinborne gave me hands-on experience with:

- Structuring a mid-sized GML codebase across objects, scripts, and rooms
- Debugging real-time systems (timing issues, collision edge cases)
- Designing tight feedback loops that make games feel responsive
- Scope management — shipping a focused prototype over an over-engineered concept

---

## Running the Project

1. Install [GameMaker Studio 2](https://gamemaker.io) (free tier works)
2. Clone this repo
3. Open `Ruinborne.yyp` in GameMaker Studio
4. Hit **Run** (F5)
